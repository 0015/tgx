Detailed Description

template<typename color_t_im, typename color_t_tex, typename ZBUFFER_t, typename BLEND_OP = void>
struct tgx::RasterizerParams< color_t_im, color_t_tex, ZBUFFER_t, BLEND_OP >

Uniform parameters passed to the shader (for internal use).

Structure that holds the 'uniform' parameters (in opengl sense) passed to the triangle rasterizer and then the shader when doing 3D rendering.
